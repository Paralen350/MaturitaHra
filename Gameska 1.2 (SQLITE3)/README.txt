Tato hra je určena pouze pro účely maturitního projektu školy SPŠ Kladno, Jana Palacha 1840, studentem třídy IT4 Lukášem Kaftanem.
Ve visual studiu by měla být hlavní složka "Gameska 1.2 (SQLITE3)", za těchto podmínek by vše mělo fungovat, pokud se hra zapne, ale nic se nebude načítat, tak přesuňte všechny levely "level(číslo)_data" do adresáře nad "Gameska.py" (nebo se podívejte do konzole, kde je problém :D).


This game is intended only for the purposes for a school project for SPŠ Kladno, Jana Palacha 1840, by Lukáš Kaftan, a student of IT4 class.
In visual studio, the main folder should be "Gameska 1.2 (SQLITE3)", eveything should work correctly under these conditions, however, if the game starts but nothing loads, move all levels "level(number)_data" to the directory above "Gameska.py" (or look into the console, to see where the problem is :D).
